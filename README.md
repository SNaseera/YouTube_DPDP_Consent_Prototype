# Supervised Setup — DPDP Consent Prototype

A clickable prototype of YouTube's consent flow for under-18 users, aligned to India's DPDP Act, 2023.

## How to Use

1. Download `index.html` from this folder
2. Open it in any web browser (double-click the file)
3. Click through the app — each button takes you to the next step of the flow

No installation, setup, or internet connection required.

## Flow Covered

Sign up → Age detected → Parent notified → Parental consent (OTP + privacy defaults) → Confirmed → Supervised home
